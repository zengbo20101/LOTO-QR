## 🚀 快速开始（3分钟）

### 1. 启动后端服务

```bash
cd /Users/admin/Soft/LOTO_QR/backend
npm install
npm start
```

服务将在 **http://localhost:3000** 启动

### 2. 导入微信小程序

1. 下载 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 打开微信开发者工具 → 选择"导入项目"
3. 选择目录：`/Users/admin/Soft/LOTO_QR/miniprogram`
4. 填写项目名称 → 选择AppID（测试号即可）→ 点击导入

### 3. 配置小程序API地址

编辑 `miniprogram/app.js`，修改API地址：

```javascript
globalData: {
  apiBase: 'http://localhost:3000/api'  // 开发环境
  // apiBase: 'https://your-domain.com/api'  // 生产环境
}
```

### 4. 开始使用！

## 🎯 核心功能

### ✅ 设备管理
- ✅ 创建设备，自动生成编号（如：EQP-2026-001）
- ✅ 自动生成二维码，Base64格式，微信可直接识别
- ✅ 设备列表查看和管理

### ✅ 锁牌管理 ⭐ NEW
- ✅ 创建锁牌，自动生成编号（如：TAG-2026-001）
- ✅ 锁牌状态追踪：可用 / 已锁定
- ✅ 自动生成二维码，贴在物理锁牌上

### ✅ LOTO记录
- ✅ 扫码设备或锁牌二维码，自动跳转到LOTO记录页
- ✅ 创建LOTO记录，关联设备和锁牌
- ✅ 记录创建后，锁牌状态自动变为"已锁定"
- ✅ 解除锁定，锁牌恢复"可用"状态
- ✅ 记录权限控制：只有创建者可编辑/删除/解除

### ✅ 二维码功能
- ✅ Base64格式二维码，微信可直接扫码
- ✅ 二维码内容：`pages/loto/loto?deviceId=xxx` 或 `pages/loto/loto?tagId=xxx`
- ✅ 扫码自动跳转小程序对应页面
- ✅ 长按保存二维码到相册

### ✅ 用户权限
- ✅ JWT身份认证
- ✅ 管理员可注册新用户
- ✅ 普通用户可进行LOTO操作
- ✅ 权限验证：只有记录创建者可解除/编辑/删除

### ✅ 数据导出
- ✅ CSV格式导出LOTO记录
- ✅ 支持按设备、锁牌、时间范围筛选

## 💡 业务流程

### 上锁流程
```
1. 扫描设备/锁牌二维码
2. 自动跳转到小程序对应页面
3. 选择锁牌（可选）
4. 填写LOTO信息
5. 提交 → 锁牌状态变为"已锁定"
```

### 解锁流程
```
1. 扫描设备/锁牌二维码
2. 查看当前LOTO记录
3. 点击"解除锁定"（仅创建者可操作）
4. 锁牌状态恢复"可用"
```
